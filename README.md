# xy.js
一个可以自动生成2d函数曲线的js库
![图片](https://user-images.githubusercontent.com/103297890/178303211-d0d32e9d-c4a0-4ae4-8665-23da40588bee.png)


## Documentation
### 基本使用
下载
```js
npm i xy.js
```
```js
// 引入xy.js
import XY from "xy.js"

// 初始化，并设置尺寸大小
const _y = new XY()

// 添加函数曲线
_y.push("Math.sin(x)")
